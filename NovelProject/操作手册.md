## 常用命令

### 一键写下一章（最常用）

```powershell
# 最简
python auto_write.py --novel "文明升阶"

# 带目标
python auto_write.py --novel "文明升阶" --goal "主角发现末日真相"

# 完整版
python auto_write.py --novel "文明升阶" --goal "你的目标" --update-characters --update-hooks
```

### 指定章节号重写

```powershell
python generate_chapter.py --novel "文明升阶" --chapter 11 --goal "写作目标"
```

### 调试：只更新上下文/人物/伏笔（不生成章节）

```powershell
python auto_write.py --novel "文明升阶" --skip-chapter --update-characters --update-hooks
```

### 单独调用子脚本

```powershell
python build_context.py --novel "文明升阶"
python generate_ideas.py --novel "文明升阶"
python build_characters.py --novel "文明升阶"
python update_hooks.py --novel "文明升阶"
python analyze_scene_type.py --file "Novels/文明升阶/04_Chapters/第10章 全球公布.md"
```

### 手动更新章节规划表标题

```powershell
python update_plot_outline.py --novel "文明升阶" --chapter 16 --title "星海独白"
```

### 删除并重写某章

```powershell
Remove-Item "Novels/文明升阶/04_Chapters/第11章 暗流涌动.md"
python generate_chapter.py --novel "文明升阶" --chapter 11 --goal "新目标"
```

## auto_write.py 参数说明

| 参数 | 说明 |
|------|------|
| `--novel` | 必填，小说文件夹名 |
| `--goal` | 可选，本章写作目标 |
| `--with-ideas` | 可选，先生成灵感 |
| `--update-characters` | 可选，生成后更新人物笔记 |
| `--update-hooks` | 可选，生成后更新伏笔表 |
| `--skip-chapter` | 可选，跳过生成章节（调试用） |

---

## generate_chapter.py 参数说明

| 参数 | 说明 |
|------|------|
| `--novel` | 必填 |
| `--chapter` | 可选，指定章节号（不填则自动取最大章号+1） |
| `--goal` | 可选，写作目标 |
| `--model` | 可选，默认 deepseek-chat |

---

## 执行顺序

`auto_write.py` 按以下顺序调用：

1. `build_context.py` — 生成上下文（必须成功，否则终止）
2. `generate_ideas.py` — 仅当 `--with-ideas`
3. `generate_chapter.py` — 核心生成
4. `build_characters.py` — 仅当 `--update-characters`
5. `update_hooks.py` — 仅当 `--update-hooks`

任一子脚本失败则终止整个流程。
